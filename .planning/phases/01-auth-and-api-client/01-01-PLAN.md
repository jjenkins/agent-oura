  <action>
Create `.claude/skills/oura/scripts/auth.mjs` implementing the complete OAuth2 flow, token storage, and refresh logic. The file must export these functions: `initAuth`, `readTokens`, `saveTokens`, `showStatus`.

**Constants (top of file):**
- `CLIENT_ID` = placeholder string `'YOUR_CLIENT_ID'` with env var override: `process.env.OURA_CLIENT_ID || 'YOUR_CLIENT_ID'`
- `CLIENT_SECRET` = placeholder string `'YOUR_CLIENT_SECRET'` with env var override: `process.env.OURA_CLIENT_SECRET || 'YOUR_CLIENT_SECRET'`
- `PORT` = `8910` (per D-01)
- `REDIRECT_URI` = `'http://localhost:8910/callback'`
- `AUTH_URL` = `'https://cloud.ouraring.com/oauth/authorize'` (browser-facing, per Pitfall 4)
- `TOKEN_URL` = `'https://api.ouraring.com/oauth/token'` (server-to-server, per Pitfall 4)
- `API_BASE` = `'https://api.ouraring.com/v2'`
- `SCOPES` = `'personal daily heartrate workout tag session spo2'` (per D-03)
- `TIMEOUT_MS` = `120_000` (per D-04)
- `TOKEN_DIR` = `join(homedir(), '.oura')` using `os.homedir()` and `path.join()`
- `TOKEN_PATH` = `join(TOKEN_DIR, 'tokens.json')` (per D-13)
- `TOKEN_TMP` = `join(TOKEN_DIR, 'tokens.json.tmp')`

**Imports:**
- `createServer` from `'node:http'`
- `randomBytes` from `'node:crypto'`
- `writeFile, readFile, rename, mkdir, chmod` from `'node:fs/promises'`
- `join` from `'node:path'`
- `homedir` from `'node:os'`
- `open` from `'open'` (default import)

**Function: `initAuth()` (exported)**
1. Generate state: `randomBytes(16).toString('hex')`
2. Build auth URL using `new URL(AUTH_URL)` with searchParams: `response_type=code`, `client_id=CLIENT_ID`, `redirect_uri=REDIRECT_URI`, `scope=SCOPES`, `state=<generated_state>`
3. Call `waitForCallback(state)` to get the authorization code
4. Call `exchangeCode(code)` to get tokens
5. Call `saveTokens(tokenResponse)` to persist
6. Call `fetchPersonalInfo(tokens.access_token)` to validate (per D-05)
7. Print to stdout: `Authenticated as <email>` (per D-05/D-09 -- identity only, no scopes)
8. Return the user info object

**Function: `waitForCallback(expectedState)` (internal)**
1. Return a Promise
2. Create HTTP server with `createServer` bound to `127.0.0.1` (NOT `0.0.0.0` -- per anti-pattern in RESEARCH)
3. Handle requests to `/callback` path only; ignore others with `res.end()`
4. Extract `code` and `state` from `url.searchParams`
5. Respond to browser with HTTP 200, Content-Type text/html, body: `<html><body><h2>Success -- return to terminal.</h2></body></html>` (per D-06)
6. Call `server.close()` and `clearTimeout(timer)`
7. Validate `state === expectedState` -- reject with `'State mismatch -- possible CSRF'` if not
8. Validate code exists -- reject with `'No authorization code received'` if not
9. Resolve with code
10. After server.listen(PORT, '127.0.0.1'), attempt `open(authUrl.toString())` -- on failure, write to stderr: `Open this URL in your browser:\n${authUrl}` (per D-02)
11. Set timeout of `TIMEOUT_MS` -- on timeout, close server, reject with error. Catch handler prints: `Auth timed out. Run /oura auth to try again.` (per D-08)
12. Handle `server.on('error', ...)` -- if `EADDRINUSE`, reject with message: `Port 8910 is in use. Free the port and run /oura auth again.`

**Function: `exchangeCode(code)` (internal)**
1. POST to `TOKEN_URL` with `Content-Type: application/x-www-form-urlencoded`
2. Body (URLSearchParams): `grant_type=authorization_code`, `code=<code>`, `redirect_uri=REDIRECT_URI`, `client_id=CLIENT_ID`, `client_secret=CLIENT_SECRET`
3. If `!res.ok`, throw with status and body text
4. Return `res.json()` -- shape: `{ access_token, refresh_token, expires_in, token_type }`

**Function: `saveTokens(tokenResponse)` (exported)**
1. Build tokens object: `{ access_token: tokenResponse.access_token, refresh_token: tokenResponse.refresh_token, expires_at: Date.now() + tokenResponse.expires_in * 1000 }` (per D-15, Pitfall 5: convert expires_in to absolute expires_at)
2. `await mkdir(TOKEN_DIR, { recursive: true })` (per Pitfall 3)
3. `await writeFile(TOKEN_TMP, JSON.stringify(tokens, null, 2), 'utf8')` (per D-16)
4. `await rename(TOKEN_TMP, TOKEN_PATH)` -- atomic rename (per D-16)
5. `await chmod(TOKEN_PATH, 0o600)` -- AFTER rename (per D-14, anti-pattern note)

**Function: `readTokens()` (exported)**
1. Try to read and parse `TOKEN_PATH`
2. On failure (ENOENT or parse error), throw `new Error('NOT_AUTHENTICATED')`
3. If `Date.now() >= raw.expires_at - 60_000` (within 60s of expiry), call `refreshTokens(raw.refresh_token)` and return result
4. Otherwise return `{ access_token: raw.access_token, refresh_token: raw.refresh_token, expires_at: raw.expires_at }`

**Function: `refreshTokens(refreshToken)` (internal)**
1. POST to `TOKEN_URL` with `Content-Type: application/x-www-form-urlencoded`
2. Body (URLSearchParams): `grant_type=refresh_token`, `refresh_token=<refreshToken>`, `client_id=CLIENT_ID`, `client_secret=CLIENT_SECRET`
3. If `!res.ok`, throw `new Error('REFRESH_FAILED')` -- do NOT retry with old token (single-use, per Pitfall 1 anti-pattern)
4. Parse response as JSON
5. **CRITICAL:** Call `await saveTokens(newTokens)` BEFORE returning -- must persist new single-use refresh token to disk first (per AUTH-04)
6. Return the saved tokens object (with computed expires_at)

**Function: `fetchPersonalInfo(accessToken)` (internal)**
1. GET `${API_BASE}/usercollection/personal_info` with `Authorization: Bearer ${accessToken}`
2. If `!res.ok`, throw with status
3. Return `res.json()`

**Function: `showStatus()` (exported)**
1. Try to call `readTokens()` (which handles refresh if needed)
2. Compute `expiresInMinutes = Math.floor((tokens.expires_at - Date.now()) / 60_000)`
3. Call `fetchPersonalInfo(tokens.access_token)` to confirm live connection
4. Write to stdout as JSON: `{ authenticated: true, email: userInfo.email || 'Unknown', token_expires_in_minutes: expiresInMinutes, connection: 'ok' }` (per D-17)
5. On any error, write to stdout as JSON: `{ authenticated: false, reason: error.message }`

**CLI entry point (bottom of file):**
Parse `process.argv[2]` to determine sub-command:
- `'auth'` -> call `initAuth()`, handle errors with user-friendly messages
- `'status'` -> call `showStatus()`
- No argument or unknown -> print usage: `Usage: node auth.mjs [auth|status]`

Wrap the CLI in an immediately-invoked block that catches errors and writes them to stderr.
