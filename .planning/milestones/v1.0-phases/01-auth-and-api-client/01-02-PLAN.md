Create `.claude/skills/oura/scripts/client.mjs` implementing the authenticated HTTP client with error classification and retry logic.

**Imports:**
- `import { readTokens } from './auth.mjs';`

**Constants:**
- `API_BASE` = `'https://api.ouraring.com/v2'`
- `MAX_RETRIES` = `3`

**Function: `ouraGet(path, params = {})` (exported)**
1. Call `await readTokens()` to get current tokens (auto-refreshes if near expiry)
2. Build URL: `new URL(\`\${API_BASE}\${path}\`)`, then set searchParams from `params` object
3. Fetch with `Authorization: Bearer ${tokens.access_token}`
4. If `res.ok` (200): return `await res.json()`
5. Error classification:

   **429 -- Rate Limited (ERR-01):**
   - Read `Retry-After` header: `parseInt(res.headers.get('Retry-After') ?? '60', 10)`
   - Throw: `Object.assign(new Error('RATE_LIMITED'), { retryAfter })`

   **401 -- Auth Expired (ERR-02):**
   - Throw: `new Error('AUTH_EXPIRED')`
   - The caller (ouraGetWithRetry) handles retry-after-refresh logic

   **403 -- Forbidden (ERR-02/ERR-03):**
   - Parse response body: `const body = await res.json().catch(() => ({}))`
   - Extract detail: `const detail = (body.detail ?? '').toLowerCase()`
   - If detail includes `'membership'` or `'subscription'`: throw `new Error('MEMBERSHIP_REQUIRED')`
   - Otherwise: throw `new Error('AUTH_FORBIDDEN')`

   **426 -- Upgrade Required (ERR-03):**
   - Throw: `new Error('APP_UPDATE_REQUIRED')`

   **Other errors:**
   - Throw: `Object.assign(new Error('API_ERROR'), { status: res.status, body: await res.text().catch(() => '') })`

**Function: `ouraGetWithRetry(path, params = {}, attempt = 0)` (exported)**
1. Try `await ouraGet(path, params)`
2. Catch errors and handle:

   **RATE_LIMITED (ERR-01):**
   - If `attempt < MAX_RETRIES`:
     - Calculate delay: `Math.min((err.retryAfter ?? 2 ** attempt) * 1000, 30_000)`
     - Write to stderr: `Rate limited -- retrying in ${delay / 1000}s (attempt ${attempt + 1}/${MAX_RETRIES})...\n`
     - `await new Promise(r => setTimeout(r, delay))`
     - Return `ouraGetWithRetry(path, params, attempt + 1)`
   - If attempts exhausted: throw error with message `Rate limit exceeded after ${MAX_RETRIES} retries. Try again in a few minutes.`

   **AUTH_EXPIRED (ERR-02):**
   - If `attempt === 0`:
     - Write to stderr: `Access token expired -- refreshing...\n`
     - Call `await readTokens()` (forces refresh since we know token is expired)
     - Return `ouraGetWithRetry(path, params, attempt + 1)` (retry with fresh token)
   - If `attempt > 0`: throw error with message `Authentication failed. Run /oura auth to re-authenticate.`

   **NOT_AUTHENTICATED:**
   - Throw error with message: `Not authenticated. Run /oura auth to connect your Oura account.`

   **REFRESH_FAILED:**
   - Throw error with message: `Token refresh failed. Run /oura auth to re-authenticate.`

   **MEMBERSHIP_REQUIRED (ERR-03):**
   - Throw error with message: `This data requires an active Oura membership. Check your subscription at https://ouraring.com.`

   **APP_UPDATE_REQUIRED (ERR-03):**
   - Throw error with message: `Please update your Oura app to the latest version to access this data.`

   **All other errors:** re-throw as-is

**Function: `formatError(error)` (exported)**
A helper that maps error codes to user-friendly messages (same messages as above but usable by SKILL.md callers):
- `RATE_LIMITED` -> `'Rate limited by Oura API. Try again in a few minutes.'`
- `AUTH_EXPIRED` -> `'Authentication expired. Run /oura auth to re-authenticate.'`
- `NOT_AUTHENTICATED` -> `'Not authenticated. Run /oura auth to connect your Oura account.'`
- `REFRESH_FAILED` -> `'Token refresh failed. Run /oura auth to re-authenticate.'`
- `MEMBERSHIP_REQUIRED` -> `'This data requires an active Oura membership.'`
- `APP_UPDATE_REQUIRED` -> `'Please update your Oura app to the latest version.'`
- `AUTH_FORBIDDEN` -> `'Access denied. Try running /oura auth to re-authenticate with all scopes.'`
- default -> `'Oura API error: ' + error.message`
