### Integration Issues

**INT-01 (low): Orphaned `formatError` export**
`client.mjs` exports `formatError` but no script imports it. Error messaging works through `ouraGetWithRetry` inline. Either wire it into consumer catch blocks or remove the export.

**INT-02 (medium): Dashboard auth failure masked as sync message**
When unauthenticated, `dashboard.mjs` shows "Today's data hasn't synced yet" instead of an auth error. Mitigated by SKILL.md's mandatory `auth.mjs status` pre-check, but the script itself provides no diagnostic. Consider checking error types in `extractRecord()`.

**INT-03 (low): Ring configuration empty array misclassified**
`profile.mjs` gets `DATA_NOT_SYNCED` for empty `ring_configuration` — wrong message for a device pairing issue. Edge case (any paired ring has records), but produces wrong UX for the affected user.
